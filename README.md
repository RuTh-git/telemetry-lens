# Telemetry Lens

A full-stack AI-powered telemetry analysis tool for Formula 1 data.

Users can:
- Load historical F1 sessions
- Compare driver telemetry
- Analyze lap performance
- Ask an AI assistant to explain race insights

## Tech Stack

Backend
- Python
- FastAPI
- FastF1
- Pandas

Frontend
- React
- Vite
- Plotly

AI
- OpenAI API

## Features (Planned)
